## 🧪 테스트 케이스: 대시보드 최근 목록 클릭 기능 추가 및 이미지 표시 수정
**이슈**: #552

### ✅ 1. 최근 가입 회원 클릭 이동

- [ ] 대시보드 최근 가입 회원 목록에서 회원 행 클릭 시 `/admin/members/{memberId}` 상세 페이지로 이동 확인
- [ ] 이동한 상세 페이지에서 해당 회원 정보 정상 표시 확인
- [ ] 커서가 pointer로 변경되는지 확인 (hover 효과)

### ✅ 2. 최근 등록 물품 클릭 이동

- [ ] 대시보드 최근 등록 물품 목록에서 물품 행 클릭 시 `/admin/items/{itemId}` 상세 페이지로 이동 확인
- [ ] 이동한 상세 페이지에서 해당 물품 정보 정상 표시 확인

### ✅ 3. 이미지 표시

- [ ] 프로필 이미지가 있는 회원 — 프로필 이미지 정상 표시 확인
- [ ] 프로필 이미지가 없는 회원 — 기본 이미지(`default-150x150.png`) 표시 확인
- [ ] 이미지가 있는 물품 — `itemImages[0]` 첫 번째 이미지 정상 표시 확인
- [ ] 이미지가 없는 물품 — 기본 이미지 표시 확인
- [ ] 이미지 URL이 만료/깨진 경우 — `onerror` fallback으로 기본 이미지 표시 확인

### ⚠️ 4. 엣지 케이스

- [ ] 회원 목록이 비어 있을 때 "등록된 회원이 없습니다" 메시지 정상 표시 확인
- [ ] 물품 목록이 비어 있을 때 "등록된 물품이 없습니다" 메시지 정상 표시 확인
- [ ] 닉네임이 없는 회원 — '익명' 표시 확인
- [ ] 판매자 닉네임 정상 표시 확인 (`member.nickname` 경로)
