# 대시보드 최근 목록 클릭 기능 추가 및 이미지 표시 수정

## 개요

관리자 대시보드의 최근 가입 회원 목록과 최근 등록 물품 목록이 단순 나열만 되어 있고 클릭 시 아무런 동작이 없던 문제를 수정하였다. 각 행 클릭 시 상세 페이지로 이동하도록 연결하고, 이미지 필드명 오류 및 이미지 로드 실패 시 fallback 처리도 함께 개선하였다.

## 변경 사항

### 클릭 이동 연결
- `dashboard.html`: 회원 행 클릭 시 `/admin/members/{memberId}` 상세 페이지로 이동
- `dashboard.html`: 물품 행 클릭 시 `/admin/items/{itemId}` 상세 페이지로 이동
- 각 `<tr>`에 `cursor-pointer hover` 클래스 및 `onclick` 이벤트 추가

### 이미지 표시 수정
- `dashboard.html`: 물품 이미지 필드 `mainImageUrl` (존재하지 않는 필드) → `itemImages[0].imageUrl` 로 수정
- `dashboard.html`: 회원 프로필 이미지 `profileUrl` null/빈 문자열 방어 처리 추가
- 이미지 로드 실패(`onerror`) 시 기본 이미지(`/assets/img/default-150x150.png`)로 교체

### 판매자 닉네임 경로 수정
- `dashboard.html`: `item.sellerNickname` → `item.member?.nickname` 우선 조회 후 fallback 처리

## 주요 구현 내용

API 응답이 `Page<Member>`, `Page<Item>` Entity 직접 반환 구조이므로 백엔드 수정 없이 프론트(Thymeleaf 템플릿 JS)에서만 수정. `memberId`, `itemId`는 UUID 타입으로 응답에 포함되므로 별도 API 변경 불필요.

## 주의사항

- `itemImages` 배열이 비어 있는 물품은 기본 이미지로 표시됨
- 클릭 이동 경로(`/admin/members/{id}`, `/admin/items/{id}`)가 실제 라우팅과 일치하는지 확인 필요
