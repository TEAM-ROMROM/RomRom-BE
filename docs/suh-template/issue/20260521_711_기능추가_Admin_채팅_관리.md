📝 현재 문제점
---

- 채팅(ChatRoom) 도메인은 현재 admin API가 **0개**다.
- 부적절한 메시지/허위 거래 채팅이 신고되어도 운영자가 채팅 내용을 들여다보거나 메시지를 삭제할 수단이 없다.
- 신고 처리 시 채팅 증거를 확인할 방법이 없어, 신고 사유의 신뢰성을 판단하기 어렵다.

🛠️ 해결 방안 / 제안 기능
---

채팅방 도메인에 admin 조회/관리 API 신규 추가.

- 채팅방 목록 조회: 참여자 검색, 신고된 채팅방 필터, 거래 연결 채팅방 필터
- 채팅방 상세: 참여자 목록 + 거래 정보 + 최근 메시지 N건
- 메시지 조회 (admin only): 페이지네이션 메시지 히스토리
- 부적절 메시지 soft delete: 메시지 단위로 노출 차단 (실데이터 보존, 다른 사용자에게 안 보임 처리)
- 채팅방 강제 종료: 거래 강제 취소와 연동될 수도 있음
- 사용자 강제 퇴장: 특정 회원을 채팅방에서 강제 제거

운영 액션 모두 SanctionHistory에 기록 권장.
