⚙️[기능추가][관리자][AI프롬프트] 가격 표시용 JSON 프롬프트를 실시간으로 동적 설정할 수 있는 UI 및 로직 추가

📝 현재 문제점
---

- 현재 AI 프롬프트에서 가격 표시에 사용되는 JSON 형식이 코드에 하드코딩되어 있어, 변경 시 배포가 필요함
- 관리자가 가격 표시 관련 프롬프트를 실시간으로 수정하거나 테스트할 수 있는 방법이 없음
- 프롬프트 변경 사항을 즉시 반영하려면 서버 재배포가 필요하여 운영 효율이 떨어짐

🛠️ 해결 방안 / 제안 기능
---

- 관리자 페이지 설정 화면에 AI 프롬프트(가격 표시용 JSON 포함)를 편집할 수 있는 전용 UI 추가
- 프롬프트를 실시간으로 수정하고 즉시 적용할 수 있는 동적 프롬프트 관리 기능 구현
- JSON 형식의 가격 표시 프롬프트를 별도로 관리하여 독립적으로 수정 가능하도록 분리
- 수정된 프롬프트가 서버 재시작 없이 실시간으로 AI 호출에 반영되는 구조

⚙️ 작업 내용
---

- 관리자 페이지에 AI 프롬프트 설정 UI 추가 (텍스트 에디터 / JSON 에디터)
- 가격 표시용 JSON 프롬프트를 별도 항목으로 분리하여 관리할 수 있는 입력 폼 구성
- 프롬프트 저장/수정 API 구현 (백엔드)
- 저장된 프롬프트를 실시간으로 불러와 AI 호출 시 적용하는 로직 추가
- 프롬프트 변경 이력 관리 (선택사항)
- JSON 유효성 검증 기능 추가
