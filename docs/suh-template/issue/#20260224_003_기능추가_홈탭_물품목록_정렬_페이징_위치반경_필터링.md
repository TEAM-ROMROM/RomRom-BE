⚙️[기능추가][홈탭][물품목록] 정렬/페이징 파라미터 및 사용자 위치 반경 자동 필터링 로직 추가

📝 현재 문제점
---

- 홈탭에서 물품 목록 요청 시 sort 파라미터를 RECOMMENDED로만 보내고 있어, sortField·sortDirection·pageNumber·pageSize 등 세부 정렬/페이징 옵션이 지원되지 않음
- 사용자의 적용 범위(radiusInMeters)를 프론트에서 매번 직접 넣어서 요청하기 어려움
- 위치 기반 필터링(사용자 설정 반경 내 물품만 조회)이 자동으로 적용되지 않아, 프론트 측 부담이 큼

🛠️ 해결 방안 / 제안 기능
---

- 홈탭 물품 목록 조회 API에 sortField, sortDirection, pageNumber, pageSize 파라미터를 추가하여 프론트에서 세부 정렬/페이징 제어 가능하도록 개선
- radiusInMeters는 프론트에서 매번 전달하지 않고, 백엔드에서 해당 사용자의 설정값(사용자 적용 범위)을 자동으로 조회하여 위치 반경 필터링에 적용
- 사용자가 설정한 위치 적용 범위 내의 물품만 자동으로 탐색되도록 백엔드 로직 추가

⚙️ 작업 내용
---

- 홈탭 물품 목록 조회 API 요청 파라미터에 sortField, sortDirection, pageNumber, pageSize 추가
- 기존 RECOMMENDED sort 방식과 신규 정렬 옵션 간 호환 처리
- 사용자별 radiusInMeters 설정값을 백엔드에서 자동 조회하여 물품 필터링에 적용하는 로직 구현
- 프론트에서 radiusInMeters를 별도로 전달하지 않아도 사용자 설정 반경 기준으로 물품이 필터링되도록 처리
