⚙️[기능추가][차단][물품요청] 차단된 사용자의 물품에 요청 시 백엔드에서 차단 처리

📝 현재 문제점
---

- 회원 차단 기능은 존재하지만, 차단된 사용자의 물품에 대해 요청하기 버튼을 클릭하면 백엔드에서 별도 검증 없이 요청이 정상 처리됨
- 차단된 사용자 간에는 상호작용이 불가능해야 하나, 물품 요청 API에서 차단 여부를 확인하지 않고 있음

🛠️ 해결 방안 / 제안 기능
---

- 물품 요청 API 호출 시, 요청자와 물품 소유자 간의 차단 관계를 확인하는 검증 로직 추가
- 차단 관계가 존재할 경우 적절한 에러 응답 반환 (예: 403 또는 커스텀 에러코드)
- 양방향 차단 모두 고려 (내가 상대를 차단한 경우 + 상대가 나를 차단한 경우)

⚙️ 작업 내용
---

- 물품 요청 API에 차단 여부 검증 로직 추가
- 차단 상태에 대한 에러 응답 코드 및 메시지 정의
- 양방향 차단 관계 확인 처리
