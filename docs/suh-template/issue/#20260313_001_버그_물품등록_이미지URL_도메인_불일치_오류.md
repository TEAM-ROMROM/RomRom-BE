❗[버그][물품등록][Storage] 물품 등록 시 이미지 URL 도메인 불일치로 등록 실패

## 본문

🗒️ 설명
---

- 물품 등록 API(`/api/item/post`) 호출 시 `잘못된 파일이 요청되었습니다.` 에러 발생
- 스토리지가 기존 NAS(`http://suh-project.synology.me`)에서 MinIO S3(`https://s3.suhsaechan.kr`)로 전환되었으나, 파일 경로 추출 로직이 단일 `file.domain` 설정값으로만 startsWith 검증을 수행하여 도메인 불일치 시 예외 발생

🔄 재현 방법
---

1. 물품 등록 화면에서 이미지를 업로드
2. 업로드된 이미지 URL이 `https://s3.suhsaechan.kr/...` 형태로 반환됨
3. 해당 URL로 물품 등록 요청 시 `CustomException: 잘못된 파일이 요청되었습니다.` 발생하며 등록 실패
